# Cross-Surface Evals

This directory holds fixtures for workflows that navigate, await capability registration on the destination surface, and continue execution.

These fixtures are governed by the north-star's cross-surface execution semantics ([§7](../../Steerable-Protocol-NorthStar.md#7-architecture)) and the roadmap's eval approach ([D6](../../docs/plan/ROADMAP.md#d6--eval-approach-repo-now-format-normative-later)).

Do not put browser automation scripts or framework-specific route tests here. This directory is for portable fixture cases unless a later eval issue says otherwise.
