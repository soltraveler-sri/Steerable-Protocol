# Evals

This directory holds fixture formats and fixtures for checking whether a Steerable integration routes intent, resolves policy, supports reversibility, and handles cross-surface execution.

Eval work is governed by the north-star's conformance and eval expectations ([§8](../Steerable-Protocol-NorthStar.md#8-developer-experience-humans-and-coding-agents), [§14](../Steerable-Protocol-NorthStar.md#14-open-questions-the-honest-ones)) and by [`ROADMAP.md#d6--eval-approach-repo-now-format-normative-later`](../docs/plan/ROADMAP.md#d6--eval-approach-repo-now-format-normative-later).

Do not add live-model harnesses, API-key requirements, or generated reports here in Stage 1 unless a later issue changes the roadmap.
