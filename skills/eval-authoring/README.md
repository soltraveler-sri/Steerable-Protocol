This directory is the planned home for the eval-authoring Agent Skill: guidance for writing fixtures that exercise intent routing, policy decisions, reversibility, and cross-surface behavior.

Skill work here is governed by the north-star's eval and coding-agent expectations ([§8](../../Steerable-Protocol-NorthStar.md#8-developer-experience-humans-and-coding-agents), [§14](../../Steerable-Protocol-NorthStar.md#14-open-questions-the-honest-ones)) and [`ROADMAP.md#d5--skillkit-format-normative-for-the-kit`](../../docs/plan/ROADMAP.md#d5--skillkit-format-normative-for-the-kit).

This is not a usable skill until a later issue adds `SKILL.md` and any supporting files. Do not add fixture files or runner logic here; those belong under `evals/` when their issues call for them.
