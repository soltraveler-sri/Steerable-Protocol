# Integration Audit Skill

This directory is the planned home for the integration-audit Agent Skill: a checklist-driven review workflow that checks a Steerable integration against spec requirement IDs and conformance criteria.

Skill work here is governed by the north-star's conformance expectations ([§8](../../Steerable-Protocol-NorthStar.md#8-developer-experience-humans-and-coding-agents)) and [`ROADMAP.md#d5--skillkit-format-normative-for-the-kit`](../../docs/plan/ROADMAP.md#d5--skillkit-format-normative-for-the-kit).

This is not a usable skill until a later issue adds `SKILL.md` and any supporting files. Do not place audit results, ad hoc notes, or partial checklists here.
