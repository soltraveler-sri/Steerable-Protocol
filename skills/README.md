# Skills

This directory holds Agent Skills for coding agents that retrofit, audit, or extend Steerable integrations.

Skill work is governed by the north-star's integration-kit requirements ([§8](../Steerable-Protocol-NorthStar.md#8-developer-experience-humans-and-coding-agents)) and [`ROADMAP.md#d5--skillkit-format-normative-for-the-kit`](../docs/plan/ROADMAP.md#d5--skillkit-format-normative-for-the-kit).

Only issue-approved skill directories belong here. A usable skill appears only when its `SKILL.md` and supporting files are written by the relevant issue; ad hoc scripts and tool-specific experiments do not belong here.
