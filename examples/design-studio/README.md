Example work here is governed by the north-star's repo shape and staged plan ([§9](../../Steerable-Protocol-NorthStar.md#9-repo-shape), [§10](../../Steerable-Protocol-NorthStar.md#10-staged-plan)) and by [`ROADMAP.md#d3--the-example-app-examplesdesign-studio-repo`](../../docs/plan/ROADMAP.md#d3--the-example-app-examplesdesign-studio-repo).

Until the example-shell issue writes real app content, only this directory README belongs here. Do not add framework setup, generated assets, package files, or proto-runtime code early.
