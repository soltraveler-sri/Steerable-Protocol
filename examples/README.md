# Examples

This directory holds the Stage-1 proving ground application and, later, any issue-approved example apps. The planned canonical app lives under `design-studio/`.

Example work is governed by the north-star's repo shape and staged plan ([§9](../Steerable-Protocol-NorthStar.md#9-repo-shape), [§10](../Steerable-Protocol-NorthStar.md#10-staged-plan)) and by [`ROADMAP.md#d3--the-example-app-examplesdesign-studio-repo`](../docs/plan/ROADMAP.md#d3--the-example-app-examplesdesign-studio-repo).

Packages, reusable SDK extraction, generated assets, and extra example apps do not belong here unless a later issue explicitly adds them. Do not add app code, dependencies, or build tooling until the example-shell issue calls for them.
