# Plan record

This directory is the project's dated process record, not its current product front door.
[`ROADMAP.md`](./ROADMAP.md) is the founding staged plan; [`GROUNDING.md`](./GROUNDING.md) preserves its mission, principles, and risks.
The Stage-1 gate is recorded in [`stage1-gate-review.md`](./stage1-gate-review.md).
Its evidence includes the frozen [`dry-run rubric`](./stage1-gate-dryrun-rubric.md), the [`dry-run report`](./stage1-gate-dryrun.md), and the preserved [`retrofit patch`](./stage1-gate-dryrun-retrofit.patch).
The package-extraction gate is recorded in [`stage2-gate-review.md`](./stage2-gate-review.md).
Together these documents show what was proposed, exercised, revised, and accepted at each gate.
They contain dated paths, counts, and phase language that reflect the repository at the time.
They are deliberately not rewritten as the project evolves; use the [root README](../../README.md) and [docs index](../README.md) for current orientation.
